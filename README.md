# gc-glitches-3

a collection of glitch and drum sounds to be used on strudel from [glitch.cool](https://www.glitch.cool/) vol 3

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/gc-glitches3')
```
please support the wonderful glitch.cool collective responsible for all these samples by grabbing the full pack through [here](https://glitchdotcool.gumroad.com/l/gwf3samples)
